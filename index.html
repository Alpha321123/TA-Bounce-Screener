v10 and later: Use only corsproxy.io — safe, no security warnings. v8 and v9: Also used allorigins.win which some security software flags. v1–v7: Use corsproxy.io only or have no proxy. If you see security warnings when scanning, upgrade to v10.

Transferring data between versions

Your portfolio, paper trades, watchlist, and simulations are stored in your browser.

1. Open your current version → click Export data in sidebar
2. Save the .json file to your computer
3. Open the new version → click Import data
4. Select the saved file → all data restored ✓
